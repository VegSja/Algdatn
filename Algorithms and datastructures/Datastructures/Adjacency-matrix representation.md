# Adjacency-matrix representation
$$
a_{ij} =
\left\{
	\begin{array}{ll}
		1 \ \ if(i, j) \in E \\
		0 \ \ otherwise
	\end{array}
\right.
$$
## Attributes
>**Memory**
>$\Theta(V^2)$

> $A = A^T$

## Advantage
- Require only one bit of memory for unweighted graphs per element
- Can show connections lookup better

## Disadvantage
- Memory usage
